OpenProfiles

Welcome to OpenProfiles. This service exposes simple profile APIs used by our front-end. In this CTF challenge you're a normal user — can you find the admin's profile?

Useful endpoints

  • /api/me — returns your profile link
  • /api/profile/2 — example profile
  • /api/users — public user list (IDs)

Hint

Profile IDs are numeric and predictable. Try modifying the /api/profile/<id> path.

Good luck!
